# Back-End Server

## Installation Guide

Requirements:
- Java 21
- Node 20.11.1 LTS
- MongoDB Community Server

## Running the Back-End

1. Create a database in Compass called "soen343" with a "users" document
2. CD into the server folder
3. Build and run the project with gradle

```bash
./gradlew build
./gradlew bootRun
```
